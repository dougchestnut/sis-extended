This zip archive contains one CSV-format file for each class. The file name is the class number on SIS and the file contain enrollment data for that class. For example, the file 12345.csv contains enrollment data for the class numbered 12345 on SIS.

Each CSV-format file contains one or more lines and each line contains three entries (actually, three numbers) separated by commas.
Entry 1: a standard unix time, measured in the number of seconds since the Unix Epoch (January 1 1970 00:00:00 GMT). 
Entry 2: the number of students enrolled in the class at the time of Entry 1.
Entry 3: the number of students on the waitlist for the class at the time of Entry 1.
